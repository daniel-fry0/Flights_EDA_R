The embedded app below shows the student facing model that I have hosted online for student use. It allows students to set a task they would like to predict based on previous performance, and then enter which completed tasks they would like to use to predict that result. The model will then provide a prediction for that result and display the R2 and MSE for the prediction. However, the main purpose of the model, and when it is at its most accurate, is when it is used to predict students' final HSC result using all prior tasks. Here, the MSE is quite low and the R2 is quite high.

The notebook below is for a slightly different deployment of the model, which allows the user to select if they would like to predict for a cohort from CSV, or if they are an individual and would like to do an individual prediction. The functionality and logic is otherwise the same.
